# 💰 Expense Tracker App (Data Science Project)

## 📌 Overview

This project is a data-driven Expense Tracker that helps users monitor, analyze, and visualize their financial behavior.

## 🚀 Live Demo

👉 https://data-science--expense-tracker-app-ujcab5dsvt5z4qhlribkfm.streamlit.app/

## 🎯 Problem Statement

Managing daily expenses manually is difficult and often leads to overspending due to lack of visibility.

## 💡 Solution

This app:

* Tracks expenses
* Categorizes spending
* Visualizes trends
* Provides insights for better financial decisions

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Streamlit

## 📊 Features

* Category-wise spending analysis
* Monthly trend visualization
* Expense vs Income comparison
* Interactive dashboard with filters
* Real-time insights
